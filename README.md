系級班級：資工1A

成員：杜承翰（組長）、陳志豪、黃浩閔

專題題目：德州撲克

程式介紹(含UML圖)

遊戲用牌為一副牌，去除兩張鬼牌。 每人隨機發5張牌，總共2~10名玩家。玩家必須將發到手中的5張牌組成最好的牌型組合，所有的下注回合結束後，玩家將亮出手中的牌，並比較牌面，以決定最終的贏家。贏家將獲得注池中所有的籌碼。
