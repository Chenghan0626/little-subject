組別：5

系級班級：資工1A

成員：杜承翰（組長）、陳志豪、黃浩閔

專題題目：德州撲克

程式介紹(含UML圖)

遊戲人數可由玩家輸入設定為2~10人。
由玩家先下注，下注完後每人發五張牌，電腦玩家隨機下注，所有玩家各自比較牌面，輸牌便損失注金，最終的贏家將獲得注池中所有的籌碼。
